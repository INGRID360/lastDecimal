# Last Decimal

A small library that increase or decrease the last decimal

## Installation

`npm install lastdecimal`  
`yarn add lastdecimal`

## Usage

    var numFormatter = require('@jdaudier/number-formatter');

    var formattedNum = numFormatter(35666);
